LoopLlama: a web application to loop and slow down YouTube videos.
  • Create favorites to quickly switch among frequently watched videos.
  • Set marks to jump to key spots in the videos.
  • Favorites and video settings are remembered.
  • Entirely keyboard driven, so no wasted time fiddling with the mouse.
  • Built for practicing music, but potentially other uses.
  • Inspired by SlowTube.
